# Grant Submission Checklist

This checklist ensures all requirements are met before submitting each grant proposal.

## General Requirements (All Grants)

- [ ] Proposal follows the specific template provided by the foundation
- [ ] Budget is realistic and well-justified
- [ ] Timeline is achievable with clear milestones
- [ ] Team qualifications are clearly stated
- [ ] Technical implementation details are comprehensive
- [ ] Impact on the ecosystem is clearly articulated
- [ ] Future roadmap is defined
- [ ] All links and references are working
- [ ] Document is free of grammatical errors
- [ ] File is in the correct format (PDF, Markdown, etc.)

## NEAR Foundation Grant

- [ ] Clear explanation of why NEAR was chosen

## Pre-Submission Review

- [ ] All team members have reviewed the proposals
- [ ] Technical accuracy has been verified
- [ ] Budget calculations are double-checked
- [ ] Timeline is realistic for the team size
- [ ] All required appendices are included
- [ ] Contact information is current and correct
- [ ] Submission portal credentials are ready
- [ ] Backup copies of all documents are saved

## Post-Submission Follow-up

- [ ] Confirmation of receipt from each foundation
- [ ] Calendar reminders for any required follow-ups
- [ ] Prepare for potential interviews or presentations
- [ ] Gather any additional information that might be requested
- [ ] Track submission status in a spreadsheet
